Latest Projects

Interactive Julia and Mandelbrot Set Visualizer

Julia and Mandelbrot and sets are intricate fractals which contain beautiful structures. You adjust the parameters of the underlying equations and explore their intricate structures with real-time zooming and panning. This is made possible through the use of GPU acceleration and computational shaders.

Profiling for Windows Graphics applications

An application which allows for the profiling and analysis of graphics-driven applications. This system gathers key metrics from any number of processes running on any number of machines such as FPS and memory usage. This data can then be visualized and analyzed on a central server.

Twitch Clip Compiler

This program automatically curates, downloads, edits and uploads to Youtube the most interesting livestream moments for a given game on Twitch. A front-end interface allows the user to fine-tune the curation process, while a backend server performs other tasks such as editing and uploading on a daily schedule.

Computational Fluid Dynamics Tutorial

This project is a working fluid simulator that takes advantage of GPU hardware acceleration. Alongside is a tutorial which explains the Navier-Stokes equations which govern fluid flow, computational methods for finding their solutions, and how to take advantage of GPU shaders to implement them.
